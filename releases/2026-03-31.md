# Release Notes — 2026-03-31

## feat: Puppeteer 서버사이드 PDF 마이그레이션 (oklab 근본 해결)

커밋: `b89857c`

### 배경
Tailwind v4는 불투명도 수정자(`bg-amber-200/60` 등)를 `color-mix(in oklab, oklch(...) %, transparent)` 형태의 CSS로 생성해 `document.adoptedStyleSheets`에 삽입합니다. 클라이언트 사이드 `html2canvas`는 `oklab` 색 공간을 파싱하지 못해 `RangeError`로 반복 크래시했습니다. 패치(insertRule 방식, getComputedStyle inline 방식)를 5회 시도했지만 모두 실패했으며, 근본 해결책으로 서버사이드 Puppeteer 방식으로 전환했습니다.

### 변경 사항

#### `api/generate-pdf.ts` (신규)
- **Vercel Serverless Function** — POST `/api/generate-pdf`
- 프론트엔드에서 직렬화된 자급자족 HTML을 받아 Puppeteer(headless Chrome)으로 렌더링 후 A4 PDF 반환
- **보안**: CORS 화이트리스트(sajuanalysis.vercel.app + localhost), `X-PDF-Token` 헤더 인증, 입력 4MB 제한, request interception으로 폰트 CDN 이외 외부 요청 차단(SSRF 방지)
- **Vercel 환경**: `@sparticuz/chromium-min` 패키지로 serverless용 Chromium 다운로드
- **로컬 환경**: 시스템 Chrome 자동 탐지 또는 `CHROME_PATH` 환경변수

#### `src/components/admin/PremiumReportPreview.tsx`
- `jsPDF`, `html2canvas` import 완전 제거
- `handlePdf` 함수를 API 호출 방식으로 전면 교체:
  1. `adoptedStyleSheets` CSS 및 `<style>` 태그 텍스트를 직렬화
  2. Google Fonts 등 폰트 CDN `<link>` 태그 포함
  3. 자급자족 HTML 생성 후 `/api/generate-pdf`에 POST
  4. PDF Blob을 로컬 다운로드 + Firebase Storage 업로드
- oklab 패치 코드(sanitizeOklabDecls, patchStyleSheets, applyColorPatch 등) **전부 삭제**

#### `tsconfig.json`
- `types`에 `"vite/client"` 추가 → `import.meta.env` 타입 오류 해소

#### `vercel.json` (이전 세션에서 설정 완료)
- `functions["api/generate-pdf.ts"]`: memory 1024MB, maxDuration 60s

### 필요 환경변수 설정

| 위치 | 변수명 | 값 |
|---|---|---|
| Vercel 대시보드 | `PDF_API_TOKEN` | 랜덤 비밀 문자열 |
| Vercel 대시보드 | `CHROMIUM_BINARY_URL` | (옵션) sparticuz chromium 바이너리 URL |
| 로컬 `.env` | `VITE_PDF_API_TOKEN` | Vercel의 PDF_API_TOKEN과 동일값 |
| 로컬 `.env` | `CHROME_PATH` | (옵션) `C:\Program Files\Google\Chrome\Application\chrome.exe` |

### 로컬 개발 방법
```bash
# Vercel CLI로 로컬 함수 서버 실행 (vercel dev가 /api 라우팅 처리)
vercel dev
```
또는 `npm run dev`만 실행 시 `/api/generate-pdf` 호출은 404를 반환합니다. PDF 기능 테스트는 `vercel dev` 필요.

### 핵심 원리
> Puppeteer = 실제 Google Chrome 엔진 → `oklch`, `oklab`, `color-mix` 완벽 지원  
> html2canvas = CSS 재구현체 → 최신 색 공간 파싱 불가
