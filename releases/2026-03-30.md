# Release Notes - 2026-03-30

## Summary
This release integrates the Liquid Glass UI kit into the application and applies a cohesive visual refresh across the main tabs. It also improves report readability and resolves PDF export margin/pagination issues.

## Highlights
- Liquid Glass kit assets integrated into `public/`.
- App-wide glass style unification in `src/App.tsx`.
- Mobile layout and touch-target polish.
- Report typography improvements with MaruBuri support.
- PDF export margin and multipage placement fixes.

## Technical Changes
- Added `playwright` dev dependency in `package.json`.
- Updated lockfile for dependency sync in `package-lock.json`.
- Updated guide page in `public/guide.html`.
- Updated global font tokens and report heading font mapping in `src/index.css`.

## Validation
