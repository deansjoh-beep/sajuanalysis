# Release Notes — 2026-04-06

관리자 전용 사주 인생네비게이션 리포트 제작 도구를 전면 재구축하였습니다.

### 신규 파일

| 파일 | 역할 |
|---|---|
| `src/components/admin/PremiumReportInputPanel.tsx` | 고객 정보 입력 폼 (Step 1) |

---

### 주요 변경 사항

#### 입력 폼 (`PremiumReportInputPanel`)
- 아코디언 제거 → 4개 섹션 **상시 표시**
  - 고객 기본정보
  - 고객 궁금사항 & 관심사
  - 인생이벤트
  - 관리자 설정
- **분석 레벨 3단계**: 초급 | 고급 | 초급+고급 병행
- 하단 **sticky 바**: "웹 보고서 생성하기" 버튼 항상 노출
- **AI 분석 타이머**: 예상 남은 시간 카운트다운 + 프로그레스 바
- **생성 중 제어**: 일시정지 / 재개 / 취소 (AbortController)

#### 리포트 프리뷰 (`PremiumReportPreview`)
- `[EASY_START]...[EASY_END]` 마커 → 스카이블루 💡 쉬운 설명 박스 렌더링
- 대운 섹션 현재 대운 표시 버그 수정 (`startYear` 기반 연도 비교)
- CoverPage 분석 레벨 `both` → `'초급+고급 병행'` 레이블 추가

#### PDF 생성 버그 수정
- 무한루프 제거 → `Math.ceil` 기반 `totalPages` for 루프
- PNG → JPEG 95% (파일 크기 절감)
- `document.fonts.ready` 대기 (한글 폰트 깨짐 방지)
- Firebase Storage 파일명 한글 안전 처리
- `printRef` div `position: relative` 추가

#### 스크롤 수정
- 전역 `overflow: hidden`에 막힌 화면 스크롤 복원
- 모든 페이지 컨테이너 → `h-full overflow-y-auto`
- 섹션 `scroll-margin-top: 52px` (sticky 네비바 가림 방지)

#### 기타 수정
- `.env.local`: `GEMINI_API_KEY` → `VITE_GEMINI_API_KEY` (Vite 접두사 수정)
- `generateLifeNavReport`에 `AbortSignal` 파라미터 추가
- `premiumOrderStore.ts` 레벨 타입: `'basic' | 'advanced' | 'both'`
- `promptBuilders.ts` `'both'` 모드 `levelNote` + `[EASY_START]` 지시 추가
- `App.tsx` Suspense fallback `h-screen` → `h-full`
- 불필요한 import 제거 (`calculateGyeok`, `getCareerFocus`)
