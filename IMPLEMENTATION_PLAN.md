# 유아이(UI) 사주 리포트 서비스 — 구현 지시서 (v2)

> **이 파일의 용도**: Claude Code가 Phase 단위로 작업을 실행하기 위한 마스터 지시서.
> 저장소 루트에 두고, 각 세션에서 "IMPLEMENTATION_PLAN.md의 Phase N을 진행해줘"로 호출한다.
> 각 Phase는 이전 Phase의 DoD(완료 기준)를 모두 통과해야 시작할 수 있다.

---

## 0. 프로젝트 컨텍스트

- **기존 자산**: https://sajuanalysis.vercel.app/ — Next.js + Vercel 배포, 만세력 산출, AI 통변(간명), 실시간 1:1 상담, 개인정보 자동 파기 구현됨
- **프로젝트 성격**: 신규 구축이 아니라 **상담형 → 결제 기반 리포트 판매형 전환·증축**
- **목표 일정**: 10월 말 사전예약 → 11월 초 정식 오픈
- **스택**: Next.js 14 (App Router), TypeScript, Vercel, Vercel Postgres(Neon), Anthropic Claude API, 토스페이먼츠(또는 포트원)

### 절대 원칙 (모든 Phase에 적용)

1. **계산과 해석의 분리**: 만세력·명리 판단(간지, 대운, 신살, 격국, 용신, 합충)은 100% 결정론적 TypeScript 코드로 구현한다. LLM에게 계산을 맡기는 코드를 발견하면 즉시 코드로 이관한다. LLM의 역할은 확정된 구조 JSON을 자연어로 푸는 것뿐이다.
2. **개인정보 최소화**: DB에 이름·전화번호·이메일·주소 컬럼을 **절대 만들지 않는다**. 저장 가능한 것은 사주 코드, 명식(간지 8자), 생성일, 주문 상태뿐. 회원가입/로그인 기능을 만들지 않는다.
3. **테스트 우선**: 만세력·규칙 엔진의 모든 함수는 단위 테스트를 동반한다. 테스트 없는 계산 로직 병합 금지.
4. **⛔ OWNER 마커**: `⛔ OWNER` 표시가 붙은 항목은 대표의 입력/결정 없이는 진행하지 않는다. 해당 지점에 도달하면 필요한 결정 사항을 요약해 보고하고 대기한다.
5. **비용 상한**: LLM 호출이 포함된 모든 기능에는 호출 횟수 상한과 타임아웃을 코드 레벨에서 강제한다.
6. **커밋 규칙**: Phase별 브랜치(`phase-0-audit`, `phase-1-report` …), 작업 단위 커밋, Phase 완료 시 DoD 체크리스트를 PR 본문에 포함.

## Phase 2 — 결제 · 사주 코드 · 산출물 증축 (8월 3주 ~ 9월 2주)

### 목표
결제 → 리포트 생성 → 사주 코드 재열람 → 자동 파기의 전 과정이 무인으로 순환하는 상용 시스템을 완성한다.

### 작업

**2-1. 데이터 계층 (`db/schema.ts`)**
- [ ] 테이블 2개만 생성 (개인 식별 컬럼 금지 원칙 재확인):
  - `codes`: id, code(8자, 예 HW-3F9K2A), myeongsik(간지 8자 + 대운수 등 재생성에 필요한 최소 파라미터), created_at, deleted_at

- [ ] 리포트 본문 저장: `reports` 테이블 또는 blob — **expires_at(72시간) 필수**, 만료 크론(`vercel cron`)으로 자동 삭제
- [ ] 즉시 삭제 API: `DELETE /api/purge?code=` → codes·orders·reports 연쇄 파기, 복구 불가 명시

**2-2. 결제 (`app/api/payment/`)**
- [ ] 토스페이먼츠 연동 (⛔ OWNER: 가맹 승인 및 키 수신 대기 — 승인 전에는 테스트 키로 개발)

## 부록 B. 세션 운영 팁

- 한 세션에는 한 Phase의 작업 묶음 하나만 요청한다 (예: "Phase 1의 1-1 만세력 보강을 진행해줘").
- 세션 시작 시 이 파일과 `docs/audit/` 최신 보고서를 컨텍스트로 제공한다.
- OWNER 항목의 결정 내용은 `docs/decisions.md`에 날짜와 함께 누적 기록하게 한다 — 이후 세션의 단일 진실 원천(single source of truth).
- Phase 완료 시 "이 Phase의 DoD를 체크리스트로 검증해줘"를 마지막 명령으로 실행한다.
