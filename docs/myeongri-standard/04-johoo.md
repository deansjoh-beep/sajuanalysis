# 4장 조후 판정

## §4.1 원칙

**§4.1.1** 조후는 자평 체계 안의 **보조 판단**이다(§1.1.1). 한열(寒熱)만 판정하며,
조습(燥濕 — 辰戌丑未의 습토·조토 구분)은 v1.5 범위에서 **명시적으로 제외**한다(v2 검토).

**§4.1.2** 계절 온도는 월지가 결정하고 시지가 보조한다. 연지·일지는 계절 온도에
영향을 주지 않으므로 조후 계산에서 제외한다(현행 "조후·운 작용력 분리 규칙" 계승).

## §4.2 온도 점수

**§4.2.1** 온도 점수 t (음수 = 한랭 방향, 양수 = 조열 방향):

| 항목 | 조건 | 가점 |
|---|---|---|
| 월지 | 亥·子·丑 | −2 |
| 월지 | 巳·午·未 | +2 |
| 시지 | 亥·子·丑 | −1 |
| 시지 | 巳·午·未 | +1 |

그 외 지지는 0. t의 범위는 −3 ~ +3.

**§4.2.2** 시간 미상 명식은 시지 항목을 생략한다(§8.3). 이때 t의 범위는 −2 ~ +2.

## §4.3 조후 상태와 극단 판정

| 조건 | 상태 | 용신 개입(§6) |
|---|---|---|
| t ≤ −3 | **극한랭** | 화(火)가 **주 용신** (§6.2.1) |
| t = −2 | 한랭 | 화(火)를 **조후 희신**으로 병기 (§6.4.2) |
| −2 < t < +2 | 평온 | 개입 없음 |
| t = +2 | 조열 | 수(水)를 **조후 희신**으로 병기 (§6.4.2) |
| t ≥ +3 | **극조열** | 수(水)가 **주 용신** (§6.2.1) |

**§4.3.1** 극단(|t| = 3)은 월지와 시지가 같은 방향의 한열 지지일 때만 성립한다.
이때만 조후가 억부·격국에 앞선다. |t| = 2 발동 시 주 용신은 억부·격국이 결정하고
조후 오행은 희신 목록에 병기만 한다. (현행 "조후 발동 시 무조건 최우선" 폐기.)

## §4.4 판정 예시

| 예시 | 명식 | 월지 | 시지 | t | 상태 |
|---|---|---|---|---|---|
| §4.4-1 | 丙戌 戊戌 戊午 丁巳 | 戌(0) | 巳(+1) | +1 | 평온 |
| §4.4-2 | 癸卯 丁巳 丙辰 己丑 | 巳(+2) | 丑(−1) | +1 | 평온 (월지·시지 상쇄) |
| §4.4-3 | 甲子 庚午 丙申 庚寅 | 午(+2) | 寅(0) | +2 | 조열 → 수를 조후 희신 병기 |
| §4.4-4 | 辛亥 庚子 甲子 乙丑 | 子(−2) | 丑(−1) | −3 | **극한랭** → 화가 주 용신 |

## §4.5 현행 v1과의 차이

- 온도 점수식(월지 ±2, 시지 ±1)과 발동 임계(|t| ≥ 2)는 현행 유지.
- 변경: 발동 시 무조건 우선 → **|t| = 3 극단만 주 용신, |t| = 2는 희신 병기**로 강등.
