# 9장 골든 명식집

## §9.1 지위와 운용

**§9.1.1** `golden-set.json`이 기계 판독 정본이며, 규칙 엔진 단위 테스트가 직접 임포트한다.
본 장의 표와 JSON의 일치는 스펙 테스트로 강제한다.

**§9.1.2** 케이스 상태는 두 가지다.
- **확정**: 기대값 전체가 기준서 조항의 수기 계산으로 채워짐 → 테스트 대상.
- **감수 대기**: 명식·커버리지만 지정됨. 규칙 엔진 구현 후 엔진 산출값을 OWNER가 감수하여
  확정으로 승격한다(엔진 값을 무감수로 넣는 순환 검증 금지) → 승격 전까지 테스트 제외.

**§9.1.3** 수치 비교 허용 오차: 점수 ±0.1 (분수 연산·반올림 차이 흡수). 등급·격명·용신 등
범주 값은 정확 일치.

**§9.1.4** 모든 명식은 생성 명식이며(실존 인물 배제 — 개인정보 원칙), 월두법·시두법 유효
조합만 수록한다.

**§9.2.2 수록 조정 기록 (2026-07-06 수기 검산 확정 시)**
- GS-10 명식 교체: 종전 庚寅 戊寅 丙午 甲午는 월간 戊 투간이 위치 우선(§5.1.2)으로 식신격이
  되어 편인격 커버리지가 불가 → 癸酉 甲寅 丙午 辛卯로 교체(월간 甲 투간 = 편인격 + 辛 재성
  노출 = 탐재괴인).
- GS-09: 시지 子가 월지 午를 상쇄(t=+1)해 "조열 발동"이 아닌 "조후 상쇄" 사례로 재분류.
  복수 투간(丁 연간·丙 월간) 위치 우선으로 편재격.
- GS-12: 실계산 비율 42.0으로 신약이 아닌 중화 → "격국 용신-상신 부재" 사례로 재분류.
  신약 커버리지는 GS-07·08·09가 담당.

**§9.2.1 보충 예정 슬롯**: 극신강 상한 경계(r≈80), 극신약 하한 경계(r≈20), 편재격,
辰戌 충이 월지에 걸린 토월 명식, 亥卯未 삼합 태그(판정 불개입 확인). 규칙 엔진 구현 중
경계를 정확히 때리는 명식을 탐색해 추가한다(추가만 허용, §1.4 규약).
