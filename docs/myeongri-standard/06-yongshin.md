# 6장 용신·희신·기신 선정

## §6.1 입력

3장 강약 판정, 4장 조후 상태, 5장 격국(격명·상신 후보·성패), 2장 오행별 세력 내역.

## §6.2 주 용신 결정 트리

**§6.2.1 [1순위] 조후 극단.** |t| = 3(§4.3)이면 조후 오행(극한랭→화, 극조열→수)이
주 용신이다. `method = '조후'`.

**§6.2.2 [2순위] 억부 — 신강·극신강 (r ≥ 60).** (2026-07-06 확정)
적군 세 오행(식상·재성·관성) 중 **원국 세력 점수가 가장 큰 오행**을 주 용신으로 한다
(뿌리 있는 억제가 실효적이라는 취지). 동점 시 서열 **관성 > 식상 > 재성**.
`method = '억부(억제)'`. 기각: 관성 고정(현행 v1), 존재 기반 서열 고정.

**§6.2.3 [2순위] 억부 — 신약·극신약 (r < 40).** (2026-07-06 확정)
**인성 오행**을 주 용신으로 한다(생아자 부조; 관인상생 논리로 희신=관성이 자동 도출됨).
비겁 채택 예외는 두지 않는다. `method = '억부(부조)'`. 기각: 아군 최강 선택, 상황별 분기.

**§6.2.4 [3순위] 격국 — 중화 (40 ≤ r < 60).** 5장 상신 후보 서열(①→②)을 따라
**원국 천간에 존재하는 첫 후보**의 오행을 주 용신으로 한다. 두 후보 모두 천간에 없으면
①후보의 오행을 주 용신으로 하되 `presentInNatal` 플래그로 부재를 표기한다
(운에서 보충해야 한다는 서술 근거). `method = '격국'`.

## §6.3 희신·기신·구신·한신 — 생극 기계 도출

**§6.3.1** 주 용신 오행에서 오행 생극표로 기계 도출한다(예외 없음, 기각안: 맥락 보정 표).

| 역할 | 정의 |
|---|---|
| 희신(喜神) | 용신을 **생하는** 오행 |
| 기신(忌神) | 용신을 **극하는** 오행 |
| 구신(仇神) | 기신을 **생하는** 오행 |
| 한신(閑神) | 나머지 한 오행 |

## §6.4 부가 출력

**§6.4.1 presentInNatal.** 용신 오행이 원국 표면(천간 4자 + 지지 본기 4자)에 존재하는지.

**§6.4.2 조후 희신 병기.** |t| = 2(§4.3)이면 조후 오행을 `johooHuisin`으로 별도 병기한다.
주 희신(§6.3.1)과 겹치면 중복 표기하지 않는다.

## §6.5 판정 예시

**[예시 §6.5-1]** 癸卯 丁巳 丙辰 己丑 (r=43.2 중화, 건록격·성격, t=+1 평온)
→ §6.2.4 격국: 상신 ①관성 — 癸 연간 존재 → **주 용신 수(관성)**, presentInNatal 참.
희신 금(재생관) / 기신 토 / 구신 화 / 한신 목.

**[예시 §6.5-2]** 丙戌 戊戌 戊午 丁巳 (r=84.2 극신강, t=+1 평온)
→ §6.2.2 억부(억제): 적군 세력 = 금 15.83, 수 0, 목 0 → 최강 금 → **주 용신 금(식상)**.
희신 토 / 기신 화 / 구신 목 / 한신 수.

**[예시 §6.5-3]** 庚申 乙酉 甲午 辛未 (r=13.3 극신약, t=+1 평온)
→ §6.2.3 억부(부조): **주 용신 수(인성)**, presentInNatal 거짓(천간에 수 없고 지지 본기도
庚·辛·丁·己 — 申 중 壬은 중기라 표면이 아님) → "운에서 보충" 서술 플래그.
희신 금(관인상생) / 기신 토(탐재괴인) / 구신 화 / 한신 목.

**[예시 §6.5-4]** 甲子 庚午 丙申 庚寅 (r=40.1 중화, 양인격·미성, t=+2 조열)
→ §6.2.4 격국: 상신 ①편관(수) — 천간 부재 → **주 용신 수(관살)**, presentInNatal 참
(子 본기 癸). 희신 금 / 기신 토 / 구신 화 / 한신 목. 조후 희신 수 — 주 용신과 동일해
별도 병기 생략(§6.4.2).

## §6.6 현행 v1과의 차이

- 현행: 조후 발동 시 무조건 조후, 아니면 50 기준 이분법으로 "신강=극아자(관), 신약=생아자(인)"
  단일 오행. 변경: 3순위 결정 트리(조후 극단 → 억부 → 격국) + 신강 억제 오행의 세력 기반 선택
  + 희신·기신·구신·한신 4역할 신설.
