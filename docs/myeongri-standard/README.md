# 명리 판단 기준서 (자평 표준)

> Phase 3-1 규칙 엔진(`lib/analysis/rules/`)의 단일 정본. 기획: `docs/myeongri-standard-plan.md`.
> 판정 지침은 2026-07-06 OWNER 문답으로 확정되었다(기획서 "확정 지침" 섹션이 결정 로그).

## 지위와 사용법

- 모든 조항은 `§장.절.항` 번호를 가진다. 규칙 엔진 코드는 해당 조항 번호를 주석으로 병기하고,
  리포트 프롬프트는 판정 근거로 조항 번호를 인용한다. **완성 후 번호 변경 금지**(추가만 허용).
- 각 장의 "판정 예시"는 그대로 단위 테스트 케이스가 된다(`golden-set.json`).
- 예시 명식은 달력이 아니라 **명식(간지 8자) 직접 지정**이다 — 규칙 엔진의 입력은 만세력 산출
  이후의 명식이므로 달력 역산이 불필요하다(단, 수록 명식은 모두 월두법·시두법 유효 조합).
- 이 기준서의 범위는 **구조 판정**(세력·강약·조후·격국·용신·희기신)까지다. 십이운성·신살·공망
  등 조견표 기반 결정 항목은 기존 만세력 파이프라인(MANSE_DATA_PIPELINE.md)이 정본이며,
  통변(자연어 해석)은 LLM 가이드라인(`src/constants/guidelines/`)의 몫이다.
