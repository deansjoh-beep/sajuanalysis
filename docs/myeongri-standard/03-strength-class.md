# 3장 신강·신약 판정

## §3.1 입력

2장의 세력 비율(반올림 전 원값, 0~100).

## §3.2 5단계 분류

**§3.2.1** 경계는 하한 포함(§1.4.2).

| 세력 비율 r | 판정 |
|---|---|
| r ≥ 80 | 극신강 |
| 60 ≤ r < 80 | 신강 |
| 40 ≤ r < 60 | 중화 |
| 20 ≤ r < 40 | 신약 |
| r < 20 | 극신약 |

**§3.2.2** 억부 방향(부조/억제)의 전환점은 별도로 두지 않는다. 용신 결정(6장)은
이 5단계 분류만 참조한다 — 신강·극신강이면 억제, 신약·극신약이면 부조, 중화면 격국 위임.
(현행 v1의 "경계 40·60 vs 전환점 50" 불일치를 해소한 규정이다.)

## §3.3 판정 예시

| 예시 | 명식 | 세력 비율 | 판정 |
|---|---|---|---|
| §3.3-1 (=§2.8-1) | 丙戌 戊戌 戊午 丁巳 | 84.2 | 극신강 |
| §3.3-2 (=§2.8-2) | 庚申 乙酉 甲午 辛未 | 13.3 | 극신약 |
| §3.3-3 (=§6.5-1) | 癸卯 丁巳 丙辰 己丑 | 43.2 | 중화 |
| §3.3-4 (=§7.4-1) | 甲子 庚午 丙申 庚寅 | 40.1 | 중화 (경계 케이스: 40 하한 포함) |

## §3.4 현행 v1과의 차이

- 등급 명칭·경계 수치(80/60/40/20)는 현행 유지. 단 점수의 의미가 "아군 가산 점수"에서
  "아군 상대 비율"로 바뀌므로 개별 명식의 판정은 달라질 수 있다.
- "중립" 명칭은 **"중화"** 로 정본화한다.
