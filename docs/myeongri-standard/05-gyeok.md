# 5장 격국 판정

## §5.1 취격(取格) 절차

**§5.1.1** 월지의 지장간(부록 A-1)을 후보로 한다.

**§5.1.2 투간 검사.** 각 지장간과 **동일한 글자**(§1.3.5)가 연간·월간·시간에 있으면 투간.
복수 투간 시 **위치 우선**으로 취격한다: **월간 > 시간 > 연간**.
(기각안: 본기>중기>여기 서열 우선, 세력 우선.)

**§5.1.3 무투간.** 투간이 하나도 없으면 **월지 본기로 취격**한다. 모든 명식은 격을 가진다.
(기각안: 격 없음(null) — 현행.)

**§5.1.4 순도 플래그.** 출력에 `transparent`(투간 취격 여부)와 `damaged`(월지 충 여부, §7.1.4)를
보존해 격의 순도를 표기한다. 순도는 격명·용신 선정을 바꾸지 않고 서술 강도에만 쓰인다.

## §5.2 격명

**§5.2.1** 취격 천간의 일간 대비 십신(§1.3.7)으로 격명을 정한다.

| 십신 | 격명 | | 십신 | 격명 |
|---|---|---|---|---|
| 비견 | **건록격** | | 편재 | 편재격 |
| 겁재 | **양인격** | | 정재 | 정재격 |
| 식신 | 식신격 | | 편관 | 편관격(칠살격) |
| 상관 | 상관격 | | 정관 | 정관격 |
| 편인 | 편인격 | | 정인 | 정인격(인수격) |

(비겁 격명 정본화는 현행 gyeokyongshin.ts 계승: 월지 본기 비견=건록, 겁재=양인.)

## §5.3 순용·역용과 상신(相神) 표 (자평진전 통설 — 2026-07-06 확정)

**§5.3.1** 상신 후보는 서열이 있으며(①이 ②에 우선), 십신 범주 기준으로 존재를 검사한다
(예: "재성" = 정재·편재 천간 모두 해당).

| 격 | 분류 | 상신 후보 | 파격 요소 |
|---|---|---|---|
| 정관격 | 순용 | ①재성 ②인성 | 상관 천간 노출 |
| 정재격·편재격 | 순용 | ①관성 ②식상 | 비겁 천간 노출 |
| 정인격·편인격 | 순용 | ①관살 ②식상 | 재성 천간 노출(탐재괴인) |
| 식신격 | 순용 | ①재성 ②편관 | 편인 천간 노출(도식) |
| 편관격(칠살) | 역용 | ①식신 ②인성 | 재성 천간 노출(재생살) |
| 상관격 | 역용 | ①인성 ②재성 | 정관 천간 노출(상관견관) |
| 건록격 | 비겁류 | ①관성 ②재성 | 재·관 전무 |
| 양인격 | 비겁류 | ①편관 ②정관 | 관살 전무 |

**§5.3.2 "노출"과 "전무"의 판정 범위.** "천간 노출" = 연간·월간·시간(일간 제외).
"전무" = 원국 천간 4자와 지지 본기 4자 어디에도 해당 십신 범주의 오행이 없는 것.

## §5.4 기본 성패 — 3상태

**§5.4.1** 판정 순서:

1. 파격 요소가 있으면 → **파격 경향**
2. (아니면) 상신 후보가 천간(연·월·시간)에 있으면 → **성격**
3. (아니면) → **미성**(상신 부재 — 운에서 상신을 만나는지가 관건이라는 서술 플래그)

**§5.4.2** 구응(救應 — 파격 후 재성격), 합살류 고급 성패 변화구는 **v2 이연**을 선언한다.
규칙 엔진은 위 3상태 외의 성패 판정을 시도하지 않는다.

## §5.5 판정 예시

**[예시 §5.5-1]** 丙戌 戊戌 **戊**午 丁巳 — 월지 戌 지장간 辛·丁·戊. 투간: 戊(월간)·丁(시간).
위치 우선 월간 → 戊 취격, 비견 → **건록격**(transparent). 성패: 재(수)·관(목)이 천간·지지
본기에 전무 → **파격 경향**.

**[예시 §5.5-2]** 庚申 乙酉 **甲**午 辛未 — 월지 酉 지장간 庚·辛. 투간: 庚(연간)·辛(시간).
위치 우선: 월간(乙, 비해당) > 시간 辛 → 辛 취격, 정관 → **정관격**(transparent).
성패: 파격 요소(상관 丁 천간 노출) 없음 / 상신 재성(토)·인성(수) 천간에 없음 → **미성**.

**[예시 §5.5-3]** 癸卯 丁巳 **丙**辰 己丑 — 월지 巳 지장간 戊·庚·丙. 천간 癸·丁·己 중 동일
글자 없음(己≠戊, 오행 동일은 투간 아님 §1.3.5) → 무투간 → 본기 丙 취격, 비견 → **건록격**
(transparent=false). 성패: 재·관 전무 아님(관 癸 연간 존재) / 상신 ①관성 癸 천간 존재 → **성격**.

**[예시 §5.5-4]** 甲子 庚午 **丙**申 庚寅 — 월지 午 지장간 丙·己·丁. 천간 甲·庚·庚 중 동일
글자 없음(丙은 일간이라 제외) → 무투간 → 본기 丁 취격, 겁재 → **양인격**(transparent=false).
월지 午가 연지 子와 충(§7) → damaged. 성패: 관살 전무 아님(子 본기 癸 존재) /
상신 편관(壬)·정관(癸) 천간에 없음 → **미성**.

## §5.6 범위 제외 선언

**§5.6.1** 종격(종강·종약·종아·종재·종살)·화격 등 **외격은 v1.5에서 인정하지 않는다**.
극신강·극신약 명식은 억부 용신(§6.2.2~3)이 담당한다. (기각안: 극단 구간 종격 인정.)
