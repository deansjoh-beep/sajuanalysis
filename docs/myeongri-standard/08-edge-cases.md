# 8장 경계·특수 케이스

## §8.1 토월(辰·戌·丑·未월)

**§8.1.1** 토월도 일반 취격 절차(§5.1)를 그대로 따른다 — 투간 우선, 무투간 시 본기(토) 취격.
"잡기격" 명칭은 도입하지 않는다(격명은 §5.2 표만 사용).

**§8.1.2** 조습 구분은 v1.5 범위 제외(§4.1.1)이므로 토월 4종은 조후 계산에서 모두 0이다
(단, 丑은 한랭 지지·未는 조열 지지로서 §4.2.1 표에 이미 포함 — 표가 우선한다).

## §8.2 절입 경계일 출생

**§8.2.1** 만세력 파이프라인의 절입경계 플래그가 선 명식도 **판정은 그대로** 수행한다
(KASI 기반 절기 시각이 분 단위로 정밀하므로 계산 결정론 유지).

**§8.2.2** 플래그가 선 경우 리포트에 "절입 경계 출생으로 출생시각 오차에 민감함" 안내
문구를 포함한다. 양쪽 월주 병기는 하지 않는다(기각안).

## §8.3 출생 시간 미상 (시주 없음)

**§8.3.1** 연·월·일 3주(6자)만으로 판정한다. 시주 관련 항목을 제외하면 세력 비율식(§2.6)이
자동 정규화하므로 별도 환산은 없다(가중치 합 80 기준).

**§8.3.2** 제외 항목: 시간·시지 세력, 시간의 투간 검사(§5.1.2는 연간·월간만), 조후 시지
가점(§4.2.2), 시지 관련 충(§7.1.3의 일지-시지 쌍).

**§8.3.3** 리포트에 "시간 미상 기준 판정" 문구와 확신도 하향 표기를 포함한다.
(기각안: 판정 거부.)

## §8.4 야자시

일주·시주 경계 정책은 D-1-1(야자시 유지, `EightChar.setSect(2)`)이 정본이며 만세력
파이프라인 소관이다. 본 기준서는 만세력이 확정한 명식을 그대로 받는다.

## §8.5 지장간 표의 정본

부록 A-1 월률분야표가 유일 정본이다. `src/utils/saju.ts`의 `hiddenStems` 상수(구성·일수
주석 포함)와 일치함을 확인했다(2026-07-06). 규칙 엔진은 이 상수를 공유하되 일수 비례
배분표(A-1의 % 열)를 추가 상수로 갖는다. 두 표의 일치는 스펙 테스트로 강제한다.
