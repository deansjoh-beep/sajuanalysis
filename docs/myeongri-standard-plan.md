## 1. 이 책의 성격 — 무엇이고, 무엇이 아닌가

**이것이다:**
- **실행 가능한 판정 스펙.** 모든 조항은 "명식 8자(+대운·세운)가 입력되면 결정론적으로 하나의 출력이 나오는" 형태로 쓴다. 독자는 사람(감수자)과 코드(규칙 엔진) 둘 다이다.
- **유파 선언문.** 격국·용신은 유파마다 답이 다르므로(D-1-6), 이 책이 곧 "우리 서비스의 유파"가 된다. 다른 유파와 갈리는 지점마다 채택안과 기각안을 명시한다.
- **테스트 픽스처.** 각 장의 판정 예시 명식은 그대로 vitest 케이스가 된다. 예시 없는 조항은 미완성으로 간주한다.

**이것이 아니다:**
- 명리 입문서·교양서가 아니다. 개념 설명은 판정에 필요한 최소한만.
- 통변(해석 문장) 지침이 아니다. 통변은 LLM 가이드라인(`src/constants/guidelines/`)의 몫이고, 이 책은 그 앞단의 **구조 판정**(강약·조후·격국·용신)까지만 다룬다. — 절대 원칙 1 "계산과 해석의 분리".

## 2. 집필 규약 (전 장 공통)

1. **조항 번호 체계**: `§장.절.항` (예: §3.2.1). 규칙 코드 주석과 A/B 벤치 근거 주입이 이 번호를 인용한다. 완성 후 번호 변경 금지(추가는 뒤에 덧붙임).
2. **조항의 4요소**: ① 정의(용어·범위) ② 판정식(표·수식·우선순위) ③ 타이브레이커(동점·경합 시 결정 규칙) ④ 예시 명식 2건 이상(경계 케이스 포함).
3. **금지 표현**: "대체로", "경우에 따라", "종합적으로 판단" 등 코드화 불가능한 서술. 예외를 두려면 예외 조건 자체를 판정식으로 쓴다.
4. **수치는 표로**: 가중치·경계값·임계치는 본문 산문이 아니라 표에만 둔다(코드 전환 시 오독 방지).
5. **예시 명식 포맷** (기계 판독 가능하게 고정):
   ```
   [예시 §2.3-1] 1984-02-17 08:30 남 (양력, 서울)
   명식: 甲子년 丙寅월 戊申일 丙辰시
   기대 출력: 득령=true, 득지=false, 득세=true, 강약점수=62, 판정=신강
   ```
6. **현행 코드 대조**: 각 장 말미에 "현행 v1 휴리스틱과의 차이" 절을 두고 유지/변경을 명시한다(§7 대조표 참조).

## 3. 장 구성 — 규칙 엔진 모듈과 1:1

| 장 | 제목 | 대응 규칙 모듈(예정) | 핵심 산출 |
|---|---|---|---|
| 1 | 총론 — 채택 유파와 판정 순서 | `rules/index.ts` (파이프라인) | 유파 선언(D-1-6 확정), 전체 판정 플로우차트, 용어 정의 |
| 2 | 오행 세력 점수화 | `rules/strength-score.ts` | 위치별 가중치 표, 통근(지장간) 반영식, 투간 가산, 득령·득지·득세 정의 |
| 3 | 신강·신약 판정 | `rules/strength-class.ts` | 5단계(극신강~극신약) 경계값, 중화 구간 처리 |
| 4 | 조후 판정 | `rules/johoo.ts` | 한열 점수식(월지·시지 가중), 발동 임계, **억부와의 우선순위 규칙** |
| 5 | 격국 판정 | `rules/gyeok.ts` | 월지 지장간 투간 원칙, 본기·중기·여기 우선순위, 복수 투간 타이브레이커, 무투간 fallback, 건록·양인 정본, 외격(종격·화격) 인정 범위 |
| 6 | 용신·희신·기신 선정 | `rules/yongshin.ts` | 억부·조후·격국용신 경합 해소 규칙, 희신·기신·구신·한신 도출식, 병약·통관 인정 여부 |
| 7 | 합·충의 판정 개입 | `rules/hapchung-effect.ts` | 합화 성립 조건, 충에 의한 월령·통근 손상 반영, 원국 vs 운(대운·세운) 적용 차이 |
| 8 | 경계·특수 케이스 | (각 모듈의 분기) | 토월(辰戌丑未) 처리, 절입 경계일, 시간 미상 명식, 지장간 표 확정 |
| 9 | 판정 예시집 (골든 명식) | `rules/*.test.ts` 픽스처 | 30~50 명식 전항목 기대 출력 |
| 부록 | 조견표 | 기존 상수 대조 | 지장간 표, 십이운성 조견표 — 현행 코드 표와 일치 확인 or 명시적 교체 |

**장 순서 = 판정 파이프라인 순서**다. 2장(점수) → 3장(분류) → 4장(조후) → 5장(격국) → 6장(용신)이 데이터 흐름이므로, 뒤 장은 앞 장의 출력만 참조할 수 있고 역참조 금지(순환 판정 방지).

### 각 장이 반드시 답해야 할 질문 (현행 코드에서 추출한 미확정 지점)

**1장 총론**
- [x] D-1-6: **자평명리 표준 확정** (2026-07-06, docs/decisions.md). 조후는 자평 체계 안의 보조 판단 — 발동·우선순위 세부는 4·6장에서 판정식으로.
- [ ] 판정 결과를 리포트에 어느 확신 수준으로 표기할지(현행 provisional 문구의 존폐).

**2장 세력 점수화**
- [ ] 위치별 가중치 — 현행: 월지 35 · 일지 15 · 나머지 각 10 (합계 100). 유지? 변경?
- [ ] 현행 코드는 **통근·투간을 점수에 반영하지 않는다** (천간·지지의 표면 오행만 봄). 지장간 통근을 반영한다면: 본기/중기/여기 계수는?
- [ ] 인성·비겁만 아군으로 치는 현행 이분법 유지? (식상·재·관의 설기 감점 도입 여부)
- [ ] 득령·득지·득세의 정확한 정의(현행 gyeokyongshin.ts의 3분해 기준 확정).

**3장 신강·신약**
- [ ] 경계값 — 현행: ≥80 극신강 / ≥60 신강 / ≥40 중립 / ≥20 신약 / 미만 극신약. 그리고 억부 용신 방향은 **50에서 뒤집힘** (경계 체계와 불일치 — 통일 필요).
- [ ] "중립(중화)" 명식의 용신 처리(현행은 50 미만이면 무조건 부조 방향).

**4장 조후**
- [ ] 현행: 월지 亥子丑/巳午未 ±2, 시지 ±1, |합|≥2에서 발동, 발동 시 **억부보다 무조건 우선**. 이 절대 우선순위를 유지하는가, 강약 극단(극신강·극신약)에서는 억부가 이기는가?
- [ ] 조습(燥濕) — 현행은 한열만 본다. 辰未戌丑의 조습 반영 여부.

**5장 격국**
- [ ] 투간 판정 범위 — 월지 지장간이 연간·월간·시간에 투출(현행). 일간 제외 확정?
- [ ] 복수 투간 시 우선순위 — 본기 > 중기 > 여기? 위치(월간 우선)?
- [ ] 무투간 시 fallback — 현행은 격 없음(null). 월지 본기로 격을 세우는가?
- [ ] 성격(成格)·파격(破格) 조건 도입 여부 — v1.5 범위에서 뺀다면 명시적으로 제외 선언.
- [ ] 종격·화격 등 외격 — 인정한다면 성립 조건(극신강·극신약 임계와 연동), 안 한다면 제외 선언.

**6장 용신**
- [ ] 억부 vs 조후 vs 격국용신이 서로 다른 오행을 가리킬 때의 서열(1장 유파 선언의 실질).
- [ ] 희신·기신·구신·한신을 용신에서 기계적으로 도출하는 식.
- [ ] 용신이 원국에 없는 글자일 때의 처리(운에서 오는 글자로 대체 서술 등).

**7장 합충 개입**
- [ ] 합이 되면 세력 점수에서 해당 글자를 빼는가, 합화 오행으로 바꾸는가(합화 성립 조건: 월령 득기 요구 여부).
- [ ] 월지가 충을 맞으면 월령 가중치(35)를 깎는가.
- [ ] v1.5 범위: 원국 내 합충만 반영하고 운과의 합충은 태그만(현행 유지)? — 범위 밖이면 명시 제외.

**8장 경계 케이스**
- [ ] 토월 戊己 처리, 辰戌丑未 월의 격국(잡기격 인정 여부).
- [ ] 절입 경계일(만세력 파이프라인의 절입경계 플래그가 선 명식)의 판정 확신도 표기.
- [ ] 시간 미상(시주 없음) 명식: 3주 판정 규칙 or 판정 거부.
- [ ] 지장간 표 — 여기·중기·본기 배분표를 부록에 확정(유파별 차이 있는 글자: 예 寅의 戊 포함 여부).

## 4. 9장 골든 명식집 — 커버리지 매트릭스

예시집은 임의 수집이 아니라 **커버리지 표를 먼저 만들고 빈칸을 채우는 방식**으로:

- 강약 5단계 × 각 1건 이상, 경계값 ±1점 명식 각 1건 (경계 검증)
- 격국: 십신격 8종 + 건록·양인 + 무투간 + (인정 시) 외격 각 1건 이상
- 조후 발동(한랭·조열) × 억부와 방향이 일치/충돌 각 1건
- 토월 4종(辰戌丑未), 절입 경계일, 합충으로 판정이 바뀌는 명식
- 실존 인물 명식 대신 **생성 명식** 사용(개인정보 원칙과 일관)

각 명식은 §2 포맷의 기대 출력 전체(점수·득령득지득세·강약·조후·격·용신·희기신)를 명기. 이 표가 그대로 `rules/*.test.ts`의 픽스처 파일(JSON)이 된다.

## 5. 포맷·저장 방식

- 리포지토리 안에서 저술: `docs/myeongri-standard/` 아래 장별 1파일 (`01-general.md` … `09-golden-set.md`, `appendix-tables.md`).
- 이유: (a) 규칙 코드 주석이 조항 번호로 상호 링크 (b) git으로 개정 이력 = 판정 변경 이력 (c) 골든 명식을 별도 JSON으로 두면 테스트가 직접 임포트.
- 수치 표는 마크다운 표로 통일. 골든 명식은 `docs/myeongri-standard/golden-set.json` 병행(문서 표와 JSON 이중화 — 스펙 테스트로 상호 일치 검증 가능).

### 2026-07-06
- **[1장] 채택 유파 = 자평명리 표준** (D-1-6). 월지 격국 중심 체계, 조후는 보조 판단. 기각: 궁통 조후 중심, 혼합.

- **[2장] 세력 점수에 지장간 통근 반영.** 지지 점수를 지장간 분해로 계산(본기·중기·여기 계수는 후속 문답). 기각: 표면 오행만(현행).
- **[2장] 위치별 가중치 = 현행 유지.** 월지 35 / 일지 15 / 연간·연지·월간·시간·시지 각 10 (합 100). v1 판정 연속성 우선.
- **[2장·3장] 세력 판정 = 아군 vs 적군 상대 비교.** 아군(비겁+인성) 세력과 적군(식상+재성+관성) 세력을 각각 합산해 비율로 강약 판정. 기각: 아군 가산식(현행).
- **[2장] 무근 천간 0.5배.** 지지 어디에도 통근하지 못한 천간은 가중치 절반(10→5). 통근 위치별 차등은 두지 않음. 기각: 위치별 차등, 통근 무시.
- **[3장] 강약 경계 = 80/60/40/20 대칭 5단계** (아군 비율 기준: ≥80 극신강 / 60~80 신강 / 40~60 중화 / 20~40 신약 / <20 극신약). 억부 방향 전환점 50과 통일 — 현행 불일치 해소.
- **[3장·6장] 중화 구간 = 격국 용신으로 위임.** 중화 명식은 억부 용신 대신 5장 격국 판정의 순용·역용 용신(상신)을 주 용신으로 채택. 억부는 보조 서술. 기각: 50 이분법(현행), 용신 보류 선언.
- **[2장] 지장간 배분 = 월률분야 일수 비례.** 각 지지의 여기·중기·본기가 주관하는 절기 일수 비례로 지지 가중치를 배분(예: 寅 = 戊7:丙7:甲16 → 약 23%:23%:53%). 월률분야표는 부록 조견표로 확정. 기각: 고정 60/25/15, 고정 70/20/10.
- **[4장] 조후 = 한열만 유지.** 조습(辰戌丑未 구분)은 v1.5 범위에서 명시적 제외 선언. 판정식(월지 ±2·시지 ±1·|합|≥2 발동)은 현행 유지.
- **[4장·6장] 조후 서열 = 극단 한열·조열만 우선.** 조후 점수 |합|=3(월지+시지 동방향 극단)일 때만 조후 오행이 주 용신, |합|=2 발동 시에는 희신으로 병기하고 주 용신은 억부·격국. 기각: 조후 무조건 최우선(현행), 항상 희신 강등.
- **[5장] 복수 투간 우선순위 = 위치 우선.** 월간 > 시간 > 연간 순으로 투간 천간을 취격(위치가 같을 수 없으므로 타이브레이커 불요). 지장간 서열(본기>중기>여기)은 후순위 참고. 기각: 본기>중기>여기 우선, 세력 우선.
- **[5장] 무투간 = 월지 본기로 취격.** 모든 명식이 격을 가짐. 단 `transparent`(투간 여부) 플래그를 별도 보존해 격의 순도를 표기. 기각: 격 없음(현행), 세력 최강 지장간.
- **[5장] 성패 = 기본 성패만.** 순용(재·관·인·식)·역용(살·상·겁·인[양인]) 격별 상신 표 + "상신이 원국 천간에 있으면 성격, 상신을 극하는 글자가 바싹 붙으면 파격 경향" 2단계만. 구응·합살류 고급 성패 변화구는 v2 이연 선언. 기각: 전면 도입, 성패 전체 제외.
- **[5장] 외격 = v1.5 제외.** 내격 10격만. 극신강·극신약은 억부 용신이 담당. 기각: 극단 구간 종격 인정.
- **[6장] 희신·기신·구신·한신 = 생극 관계 기계 도출.** 용신을 생함=희신, 용신을 극함=기신, 기신을 생함=구신, 나머지=한신. 기각: 억부 맥락 보정 표.
- **[7장] 원국 합충 = 충만 반영.** 지지충 성립 시 양쪽 지지 세력 0.5배, 월지 충이면 격 순도 하락 표기. 합은 태그만(합화 성립 판정 v2 이연). 기각: 합·충 모두 반영, 불개입(현행).
- **[7장] 운(대운·세운) 합충 = 태그만.** 운에 따른 용신·격 재판정 없음 — 격·용신은 원국 고유값. 해석은 LLM이 태그 근거로 서술.
- **[8장] 시간 미상 = 3주 판정 + 확신도 표기.** 시주 항목 제외 후 동일 비율식(상대 비교라 자동 정규화), 조후 시지 항목 생략, 리포트에 "시간 미상 기준" 명시. 기각: 판정 거부.
- **[8장] 절입 경계일 = 판정 그대로 + 안내 문구.** 계산 결정론 유지, 경계 플래그 시 민감성 안내만 추가. 기각: 양쪽 월주 병기.
- **미결 조항 6건 전건 원안 확정 (2026-07-06 문답):** ①신강 억제 용신 = 적군 최강 오행(동점 관>식>재) ②신약 부조 = 인성 고정 ③충 감쇠 = 인접만 ④상신·파격 표 = 자평진전 통설안 승인 ⑤득세 = 월지·일지 제외 아군 ≥50% ⑥골든셋 잔여 6건 = 즉시 수기 계산 확정(엔진 후 감수 방식 기각). GS-07~12 검산 완료 — GS-10 명식 교체(편인격 확보), GS-09·12 커버리지 재분류(9장 §9.2.2). **기준서 완결.**
- **[1장·부록] 자평 외 기법 4건 확정 (§1.5·A-6 반영):** ① 신살·십이운성·공망 전항목 **판정 불개입**(구조 출력 전용) ② 부록 A-6 수록 범위 = **가이드라인 요구 목록만**(12신살+양인·천을·문창·학당·괴강·원진·홍염·귀문+공망; C군 백호·삼재 등 제외) ③ **홍염살·귀문관살 즉시 구현**(`getHongyeom`·`isGwimun` + SajuAnalysis·만세력 탭 반영 — LLM 임의 판단 상태 해소) ④ **공망 = 일주 기준 단일**(SajuAnalysis·챗/리포트 프롬프트 코드 전환, 가이드라인 "일주+월주 이중" 문구 수정, 만세력 탭 3중 표시는 참고용 유지).
