# MANSE_DAEUN_PIPELINE.md

> 사주 분석 시스템의 **대운(大運)·세운(歲運) 데이터 파이프라인** 문서입니다.
> 원국 파이프라인(섹션 1~13)은 **[MANSE_DATA_PIPELINE.md](./MANSE_DATA_PIPELINE.md)** 를 참조하세요.


## 14. 대운(大運) 산출

> 소스: `src/utils/saju.ts` — `getDaeunData()` 함수

### 14-1. 순행/역행 결정

| 조건 | 방향 |
|---|---|
| 남자 + 양년생 | 순행(順行) |
| 남자 + 음년생 | 역행(逆行) |
| 여자 + 양년생 | 역행(逆行) |
| 여자 + 음년생 | 순행(順行) |

> "양년생" 판별 = 년주 천간의 음양이 `+`인 경우

### 14-2. 대운수(大運數) 계산

```
순행: 생일 → 다음 절기까지 일수
역행: 생일 → 이전 절기까지 일수

대운 시작 나이 = round(일수 / 3)  // 1일 = 3년
최솟값 = 1세
```

### 14-3. 대운 간지 진행

월주 천간·지지에서 출발하여 순행이면 +1, 역행이면 −1씩 이동합니다.  
총 10개 대운(각 10년)을 반환합니다.

### 반환 구조

```ts
[
  {
    startAge: 3,
    startYear: 1993,
    stem: '甲',
    branch: '子',
    description: '지혜와 유연함의 시기입니다...'
  },
  // ... 10개
]
```

### 14-4. 대운 십이운성(十二運星) 분석

각 대운의 **지지**를 섹션 13-2의 일간×지지 조견표에 대입하여, 해당 10년간 일간이 처한 생사 주기 단계를 파악합니다.

**적용 순서**

1. `일간(日干)` 확인 (반환 배열 index 2의 stem.hanja)
2. 각 대운 `branch` 지지를 조견표에 대입 → 운성 이름 확인
3. 운성 강약에 따라 그 10년의 에너지 방향성과 주요 테마 도출

| 운성 유형 | 해당 운성 | 10년 기간 특성 |
|---|---|---|
| **극성기** | 건록·제왕 | 자립·성취·주도권 장악, 강한 추진력, 사회적 두각 |
| **성장기** | 장생·관대 | 새 출발·배움·준비, 귀인 만남, 기반 다지기 |
| **하강기** | 쇠·병 | 보수적 안정 추구, 건강 유의, 내면 정리 필요 |
| **침잠기** | 사·절 | 에너지 소진·단절, 은둔·학문·영적 탐구로 전환 |
| **저장기** | 묘 | 재물 보관·집착 경향, 외형 성장보다 내실 다짐 |
| **회복기** | 태·양 | 새 씨앗 잉태, 타인 보살핌 속 재충전, 다음 도약 준비 |
| **감성기** | 목욕 | 도화·감수성·예술 발현, 이성 인연 복잡, 방종 주의 |

**대운 천간십성과의 복합 해석**

| 상황 | 해석 방향 |
|---|---|
| 건록·제왕 대운 + 원국 용신과 같은 오행 | 용신 에너지가 극대화되어 인생 최고 황금기 |
| 절·사 대운 + 천간이 기신(忌神) | 이중 약화 — 사업·건강 모두 침체 구간 |
| 묘 대운 + 재성(財星) 천간 | 재물 보관력 극대화, 저축·부동산 축적 유리 |
| 목욕 대운 + 관성(官星) 천간 | 명예와 이성 인연이 얽히는 복잡한 시기 |

> **AI 프롬프트 반영**: 각 대운의 운성명과 에너지 유형을 `description` 필드나 별도 `unsung` 필드로 컨텍스트에 포함하면 Gemini가 10년 단위 흐름을 구체적으로 통변할 수 있습니다.

### 14-5. 대운 십이신살(十二神殺) 분석

원국 **년지(年支)** 의 삼합 그룹(섹션 11-1)을 기준으로, 각 대운의 **지지**를 11-2 조견표에 대입하여 그 10년간 지배적인 신살을 확인합니다.

**적용 순서**

1. 원국 년지가 속한 삼합 국 확인 (인오술·해묘미·사유축·신자진)
2. 대운 지지를 해당 열에서 조회 → 신살명 확인
3. 신살 성격(길·흉·중립)과 의미로 10년 사회적 흐름 해석

| 신살 유형 | 해당 신살 | 대운에서의 10년 흐름 |
|---|---|---|
| **길운** | 장성살·반안살 | 권위·리더십·안정, 승진·인정·직위 공고화 시기 |
| **활동운** | 역마살·지살 | 이동·이사·해외·직장 변동, 분주하나 활동 반경 넓어짐 |
| **도화운** | 년살(도화) | 이성 인연·인기·대인 관계 활발, 주목받는 시기 |
| **예술운** | 화개살 | 종교·예술·명상 지향, 고독 속 창조적 활동 |
| **위기운** | 겁살·재살·망신살 | 외부 충격·관재·망신수 주의, 보수적 처신 필요 |
| **시련운** | 월살·천살 | 끊임없는 고초·불가항력적 손실, 인내와 내공 축적 구간 |
| **이동운** | 지살 | 지살은 역마에 준하나 국내 이동 성격, 환경 변화 |

**원국 신살과 대운 신살의 중첩 해석**

| 상황 | 해석 방향 |
|---|---|
| 원국 역마살 + 대운 역마살 | 극단적 이동수 — 해외 이주·전직·사업장 이전 |
| 원국 겁살 + 대운 재살 | 재물 손실과 관재가 겹치는 극히 주의 요망 구간 |
| 원국 망신살 + 대운 장성살 | 망신 요소가 권위로 눌려 경미하게 지나갈 수 있음 |
| 원국 화개살 + 대운 화개살 | 은둔·종교·예술에 완전히 몰두하는 10년 |
| 원국 도화(년살) + 대운 도화 | 이성 인연 매우 활발, 결혼 또는 스캔들 가능성 |

> **AI 프롬프트 반영**: 대운별 신살명을 `shinsal` 필드로 컨텍스트에 추가하면 AI가 "30~40대 역마살 대운에는 잦은 이직·이사가 예상됩니다"와 같이 구체적 사건 흐름을 통변합니다.

## 15. AI 상담/리포트에 전달되는 컨텍스트 형식

> 소스: `src/hooks/useChatSendAction.ts` + `src/lib/promptBuilders.ts`

```
[사주 컨텍스트]
시주: 경(庚) 오(午) - 십성: 편관/편재
일주: 임(壬) 술(戌) - 십성: 일간/편관
월주: 병(丙) 오(午) - 십성: 편재/편재
년주: 갑(甲) 오(午) - 십성: 식신/편재

[대운 컨텍스트]
3세 대운: 을묘 (정인), 13세 대운: 갑인 (식신), ...
```

두 문자열이 `buildConsultingSystemInstruction()` 또는 `buildReportSystemInstruction()`의  
`sajuContext` / `daeunContext` 파라미터로 주입되어 Gemini 모델에 전달됩니다.

## 16. 오늘 일진 / 세운 계산 (현재 시점 기준)

> 소스: `src/lib/seoulDateGanji.ts`

```ts
getTodayDayPillarKST()     // 서울 기준 오늘 일주
getCurrentYearPillarKST()  // 서울 기준 올해 연주 (세운)
```

- `Intl.DateTimeFormat('en-CA', { timeZone: 'Asia/Seoul' })`으로 서울 날짜를 추출합니다.
- `Solar.fromYmd()` → `getLunar()` → `getEightChar()`로 오늘의 간지를 계산합니다.
- 상담·리포트 시스템 프롬프트에 고정 주입되어 AI가 임의 추정하는 것을 방지합니다.

## 17. 일관성 유지를 위한 핵심 규칙 요약

| 규칙 | 내용 |
|---|---|
| 진태양시 | 경도 정보가 있을 때만 적용. 없으면 표준시 기준 사용 |
| 생시 미상 | `hour=12, minute=0` 대체 후 결과에 "생시 미상 기준" 명시 |
| 자정 전후 시주 | `setDayZero(2)` 설정 → 23:00 이후를 다음 날 시주로 처리 |
| 음력 윤달 | `month` 값을 음수로 전달 |
| 일간 판별 | 반환 배열의 `index 2` (`title: '일주'`)의 `stem` 값이 일간 |
| 대운 방향 | 성별 + 년간 음양으로 결정 (코드 변경 전 반드시 확인) |
| 세운/일진 | 반드시 `seoulDateGanji.ts`의 KST 기준 함수로 고정 조회 |
| 루나 라이브러리 | `lunar-javascript` 패키지 사용. 메서드명 변경 시 전체 파이프라인 영향 |
