# 만세력 데이터 추출 파이프라인

> 목적: 사용자의 생년월일시를 입력받아 사주 원국(四柱 原局)을 산출하고,  
> 이후 AI 상담·리포트·택일 분석에 일관되게 사용되는 만세력 데이터가  
> 어떤 과정으로 만들어지는지를 정리한 참고 문서입니다.  
> 다음 세션에서도 동일 로직으로 재현·검증할 수 있도록 작성되었습니다.

---

## 1. 사용자 입력 항목

| 항목 | 타입 | 예시 | 비고 |
|---|---|---|---|
| 생년월일 | `YYYY-MM-DD` | `1990-07-25` | 양력/음력 선택 |
| 생시 | `HH:mm` | `14:30` | 24시간제 |
| 음력 여부 | `boolean` | `false` | 양력이 기본값 |
| 윤달 여부 | `boolean` | `false` | 음력 선택 시만 유효 |
| 생시 미상 | `boolean` | `false` | true이면 정오(12:00) 대체 |
| 성별 | `'M' \| 'F'` | `'M'` | 대운 방향 계산에 사용 |
| 시간대 | `string` | `'Asia/Seoul'` | 기본값 서울 |
| 경도 | `number?` | `127.0` | 진태양시 보정에 사용 |

> 소스: `src/utils/saju.ts` → `getSajuData()` / `getDaeunData()` 함수 시그니처

## 2. 전처리 단계

### 2-1. Luxon으로 현지 시각 객체 생성

```
localDateTime = DateTime.fromObject(
  { year, month, day, hour, minute },
  { zone: timezone }
)
```

- 생시 미상(`unknownTime=true`)이면 `hour=12, minute=0`으로 고정합니다.

### 2-2. 진태양시(眞太陽時) 보정

```
// src/utils/saju.ts — applyTrueSolarTime()
offsetMinutes = 4 × (longitude - standardMeridian) + equationOfTime(dateTime)
trueSolarDateTime = localDateTime + offsetMinutes
```

- 경도가 제공되지 않을 경우 이 보정 단계를 건너뜁니다.
- 균시차(Equation of Time)는 날짜의 연중 일수(`dateTime.ordinal`)를 이용해  
  `9.87·sin(2B) − 7.53·cos(B) − 1.5·sin(B)` 공식으로 계산합니다.

### 2-3. 한국 표준시 역사 보정

```
// src/utils/saju.ts — getAdjustedTime()
```

| 기간 | 보정 |
|---|---|
| 1912-01-01 ~ 1954-03-20 | −30분 |
| 1961-08-10 이후 | −30분 |
| 서머타임 (1948–1960, 1987–1988, 5~9/10월) | −60분 |

> 위 보정은 `getAdjustedTime()`에서 분 단위 오프셋으로 반환하며,  
> 필요 시 호출부에서 `localDateTime`에 수동 적용됩니다.

## 3. 양력/음력 변환

```
// lunar-javascript 라이브러리 사용

if (isLunar) {
  solar = Lunar.fromYmdHms(year, isLeap ? -month : month, day, h, m, s).getSolar();
} else {
  solar = Solar.fromYmdHms(year, month, day, h, m, s);
}
```

- 음력 윤달은 `month`를 음수(-month)로 전달하는 방식으로 처리합니다.
- 이 단계 이후 모든 계산은 **양력 `Solar` 객체**를 기준으로 합니다.

## 4. 팔자(八字) — 사주 원국 산출

```
lunar = solar.getLunar()

eightChar.setDayZero(2)   // 23:00 이후를 다음날 시주로 전환
```

### 4-1. 네 개 기둥

| 기둥 | 메서드 | 반환 예시 | 생시 미상 처리 |
|---|---|---|---|
| 년주 (年柱) | `eightChar.getYear()` | `甲子` | 정상 반환 |

### 산출 규칙

| 관계 | 음양 동일 | 음양 상이 |

| 대상 → 일간 (대상이 극) | 편관 | 정관 |

**지지 십성**: 지지의 지장간 중 **본기(정기)**를 해당 지지의 대표 천간으로 사용합니다.

## 6. 최종 `getSajuData()` 반환 구조

`getSajuData()` 함수 반환값은 **시주 → 일주 → 월주 → 년주** 순의 배열입니다  
(즉, index 0 = 시주, index 3 = 년주).

```ts
[
  {
    title: '시주',
    stem: {
      hanja: '庚',
      hangul: '경',
      element: 'metal',
      deity: '편관',      // 십성
      western: '...',     // 영문 해설
      korean: '...',      // 한국어 해설
      careers: '...'      // 직업 표현
    },
    branch: {
      hanja: '午',
      hangul: '오',
      element: 'fire',
      deity: '편재',
      western: '...',
      korean: '...',
      careers: '...',
      hidden: '기, 정'    // 지장간 (쉼표 구분)
    }
  },
  // ... 일주, 월주, 년주 순
]
```

> 생시 미상이면 시주의 `stem.hanja = '?'`, `branch.hanja = '?'`로 설정됩니다.

## 9. 합(合)·충(沖) 판별 및 역동적 해석

사주 원국의 여덟 글자 간 상호작용을 분석하는 핵심 단계입니다.  
정지된 글자들 사이의 입체적 역학을 읽어내어 실제 삶의 사건과 심리 변화를 예측합니다.

### 9-1. 1단계: 천간(天干) 합·충 — 정신적·사회적 영향

천간은 생각·이상·사회적 타이틀 등 겉으로 드러나는 추상 영역을 상징합니다.

**천간합(天干合)** — 양(陽)과 음(陰)이 강하게 끌어당기는 결합

| 합 이름 | 짝 | 영향 |
|---|---|---|
| 갑기합(甲己合) | 갑 ↔ 기 | 합거(合去)로 기운이 묶이거나, 용신이 묶이면 능력 발휘 저하 |
| 을경합(乙庚合) | 을 ↔ 경 | 사회적 결속, 강한 의리적 유대 |
| 병신합(丙辛合) | 병 ↔ 신 | 수(水)로 화(化)하려는 지향성 |
| 정임합(丁壬合) | 정 ↔ 임 | 목(木)으로 화하려는 지향성 |
| 무계합(戊癸合) | 무 ↔ 계 | 화(火)로 화하려는 지향성 |

- 합이 발생하면 본래 오행의 고유 특성이 변질되거나 기운이 묶이는 **합거** 현상이 생깁니다.
- 용신이 합으로 묶이면 능력 발휘가 막히는 답답한 상황으로 나타납니다.

**천간충(天干沖)** — 음양이 같은 오행 상극끼리의 정면 충돌

| 충 이름 | 짝 |
|---|---|
| 갑경충(甲庚沖) | 갑 ↔ 경 |
| 을신충(乙辛沖) | 을 ↔ 신 |
| 병임충(丙壬沖) | 병 ↔ 임 |
| 정계충(丁癸沖) | 정 ↔ 계 |

- 물리적 타격보다 **가치관 전복, 극심한 정신적 스트레스, 사회적 지위·명예의 변동** 등 심리·사회적 파동으로 발현됩니다.

---

### 9-2. 2단계: 지지(地支) 합(合) — 현실적 연대 분석

지지는 구체적 현실·육체·물질 환경을 상징하므로 합의 종류가 방대하고 파급력이 즉각적입니다.

**육합(六合)** — 원형 도표 같은 위도의 지지끼리 일대일 결합

| 육합 쌍 | 키워드 |
|---|---|
| 자축(子丑) | 은밀한 정서적 유대, 사적 애착 |
| 인해(寅亥) | 남녀 관계, 강한 사적 결속 |
| 묘술(卯戌) | 관계 속 감정적 융합 |
| 진유(辰酉) | 물질적·현실적 연대 |
| 사신(巳申) | 비밀스러운 연합 |
| 오미(午未) | 따뜻하고 안정적인 결합 |

**삼합(三合)** — 생지·왕지·고지 세 글자가 모여 거대 세력 형성

| 삼합 | 구성 | 화(化)하는 오행 |
|---|---|---|
| 해묘미(亥卯未) | 亥(생)·卯(왕)·未(고) | 목(木) |
| 인오술(寅午戌) | 寅(생)·午(왕)·戌(고) | 화(火) |
| 사유축(巳酉丑) | 巳(생)·酉(왕)·丑(고) | 금(金) |
| 신자진(申子辰) | 申(생)·子(왕)·辰(고) | 수(水) |

- 두 글자만 있는 **반합(半合)**도 유의미하게 살핍니다.
- 공통 목표를 위한 직업적·사회적 결합, 뚜렷한 목적 지향성으로 해석합니다.

**방합(方合)** — 같은 방위·계절 지지끼리의 결합

| 방합 | 구성 | 계절 |
|---|---|---|
| 인묘진(寅卯辰) | 봄 방위 | 목(木) |
| 사오미(巳午未) | 여름 방위 | 화(火) |
| 신유술(申酉戌) | 가을 방위 | 금(金) |
| 해자축(亥子丑) | 겨울 방위 | 수(水) |

- 지연·학연·혈연 기반의 원초적이고 배타적인 소속감으로 작용합니다.
- 삼합보다 **맹목적이고 강한 응집력**을 보입니다.

**암합(暗合)** — 지장간 내부의 숨겨진 결합

- 겉으로 드러난 지지끼리의 결합이 아닌, **지장간 본기끼리** 또는 **천간과 지장간**이 몰래 결합하는 형태입니다.
- 표면적으로 알 수 없는 내밀한 거래, 숨겨진 자금, 비밀스러운 애정사를 추론하는 결정적 단서로 해석합니다.

---

### 9-3. 3단계: 지지(地支) 충(沖) — 현실적 붕괴·혁신 분석

원형 도표 상 180도 정반대 지지끼리의 물리적 전면 충돌(육충)입니다.

| 충 유형 | 충 쌍 | 특성 및 발현 |
|---|---|---|
| **생지충** | 인신충(寅申), 사해충(巳亥) | 폭발적 기운 충돌. 새 일 시작 단계의 격렬한 마찰, 잦은 역마(이동·이사·직장 변동) |
| **왕지충** | 자오충(子午), 묘유충(卯酉) | 육충 중 파괴력 최강. 타협 불가, 뚜렷한 가시적 붕괴·급격한 단절 |
| **고지충** | 진술충(辰戌), 축미충(丑未) | 토(土)끼리 충돌이지만 내부 지장간 봉인 해제 → 수화(水火)·금목(金木)의 대리전, 매우 복잡한 양상 |

---

### 9-4. 4단계: 충합(沖合) 종합 득실 평가

합은 길(吉), 충은 흉(凶)이라는 이분법을 배제하고 사주 전체 균형에 미치는 득실을 평가합니다.

| 상황 | 해석 방향 |
|---|---|
| 방합·삼합이 과도하여 사주에 병(病)이 되는 경우 | 오히려 그 합을 깨는 충이 필요 — 충을 긍정적으로 해석 |
| 지지충이 정체된 삶의 틀을 부수는 경우 | 강제적 혁신·변화의 긍정적 기폭제로 통변 |
| 용신이 합거되어 묶인 경우 | 능력 발휘 저하로 해석, 대운·세운에서 해소 시점 파악 |

## 10. 공망(空亡) 분석

공망이란 60갑자(甲子) 순환에서 10개의 천간이 12개의 지지를 짝지을 때 짝을 찾지 못하고 빠지는 2개의 지지입니다.  
해당 지지가 놓인 자리의 **기운이 텅 빈(空) 것처럼 현실에서 온전히 발현되지 못하고 공허하거나 지연**되는 현상으로 해석합니다.

### 10-1. 공망 계산 방법

**기준: 일주(日柱)의 천간**을 이용해 해당 일주가 속하는 순(旬)을 확인한 뒤, 그 순에서 짝이 없는 두 지지를 공망으로 확정합니다.

| 순(旬) | 일주 범위 | 공망 지지 |
|---|---|---|
| 甲子旬 | 갑자(甲子) ~ 계유(癸酉) | 술(戌)·해(亥) |
| 甲戌旬 | 갑술(甲戌) ~ 계미(癸未) | 신(申)·유(酉) |
| 甲申旬 | 갑신(甲申) ~ 계사(癸巳) | 오(午)·미(未) |
| 甲午旬 | 갑오(甲午) ~ 계묘(癸卯) | 진(辰)·사(巳) |
| 甲辰旬 | 갑진(甲辰) ~ 계축(癸丑) | 인(寅)·묘(卯) |
| 甲寅旬 | 갑인(甲寅) ~ 계해(癸亥) | 자(子)·축(丑) |

> 예시: 일주가 **壬戌(임술)**이면 甲子旬에 속하므로 공망은 **술(戌)·해(亥)**. 단, 일지(戌) 자체가 공망에 해당함 → 일지 공망.

### 10-2. 공망 지지가 위치한 기둥과 육친(六親) 의미

공망 지지가 원국의 어느 기둥(지지)에 해당하는지에 따라 영향을 받는 육친이 달라집니다.

| 기둥 | 지지 자리 | 대표 육친 | 공망 시 해석 |
|---|---|---|---|
| 년주 년지 | 年支 | 조부모, 가문 배경, 초년 환경 | 가문의 지원이 약하거나 조상 인연이 희박 |
| 월주 월지 | 月支 | 부모(특히 어머니), 형제, 직업 기반 | 부모 인연 박약, 직업 기반 불안정 가능성 |
| 일주 일지 | 日支 | 배우자, 결혼 (**가장 민감한 자리**) | 배우자 인연 지연·이별수, 결혼 생활의 공허감 |
| 시주 시지 | 時支 | 자식, 부하, 노년 결실 | 자식 인연 박약, 말년 고독 가능성 |

### 10-3. 공망 지지의 십성(十星) 분석

공망 지지의 본기를 기준으로 산출된 십성이 무엇인지 파악하면 **어떤 사회적 역학 관계가 현실에서 온전히 작동하지 못하는지** 특정할 수 있습니다.

| 공망에 해당하는 십성 | 현실적 공허 영역 |
|---|---|
| 비견·겁재 | 형제·동료·경쟁자와의 관계가 실질적 지지로 이어지지 못함 |
| 식신·상관 | 표현력·창의성·자식 복이 현실 결실로 연결되기 어려움 |
| 편재·정재 | 재물이 손에 쥐어지지 않거나, 아버지 인연이 박약 |
| 편관·정관 | 명예·직위·권력이 안정적으로 유지되기 어려움 (특히 여성의 경우 배우자 인연 약화) |
| 편인·정인 | 학문·자격·어머니 인연이 허전하거나 정서적 지지가 부족 |

### 10-4. 공망 해석 원칙

| 원칙 | 내용 |
|---|---|
| 공망 = 절대 흉이 아님 | 현실화가 지연되거나 정신적·추상적 영역으로 승화될 수 있음 |
| 공망 + 충(沖) | 공망이 충되면 공망이 해소되어 현실 발현 가능성이 열림 |
| 공망 + 합(合) | 공망이 합되면 공망의 허(虛)한 기운이 보완되어 인연이 회복되기도 함 |
| 대운·세운에서 공망 해소 | 공망 지지와 충·합이 되는 운이 들어오는 시기에 관련 사안이 가시화됨 |
| 일지 공망 주의 | 배우자 자리 공망은 가장 민감하게 살피며, 단정보다 경향성으로 안내 |

## 11. 십이신살(十二神殺) 판별

12신살은 삼합(三合) 국(局)을 기준으로 원국의 각 지지에 배속되는 12가지 기운으로,  
**년지(年支) 기준**으로 보면 사회·환경·운명적 흐름을, **일지(日支) 기준**으로 보면 개인 성향·배우자·대인 관계를 살핍니다.  
두 기준을 동시에 적용하여 겹치는 신살을 우선 해석합니다.

### 11-1. 삼합 그룹 분류

먼저 년지(年支) 또는 일지(日支)가 어느 삼합 국에 속하는지 확인합니다.

| 삼합 국 | 해당 지지 |
|---|---|
| 인오술(寅午戌) — 火局 | 年支 또는 日支가 寅·午·戌 중 하나 |
| 해묘미(亥卯未) — 木局 | 年支 또는 日支가 亥·卯·未 중 하나 |
| 사유축(巳酉丑) — 金局 | 年支 또는 日支가 巳·酉·丑 중 하나 |
| 신자진(申子辰) — 水局 | 年支 또는 日支가 申·子·辰 중 하나 |

### 11-2. 12신살 조견표

기준 지지(년지 또는 일지)가 속한 삼합 국 열에서 원국 각 지지에 해당하는 신살을 찾습니다.

| 신살 | 인오술(寅午戌) 생 | 해묘미(亥卯未) 생 | 사유축(巳酉丑) 생 | 신자진(申子辰) 생 |
|---|---|---|---|---|
| 겁살(劫殺) | 亥 해 | 申 신 | 寅 인 | 巳 사 |
| 재살(災殺) | 子 자 | 酉 유 | 卯 묘 | 午 오 |
| 천살(天殺) | 丑 축 | 戌 술 | 辰 진 | 未 미 |
| 지살(地殺) | 寅 인 | 亥 해 | 巳 사 | 申 신 |
| 년살(年殺) | 卯 묘 | 子 자 | 午 오 | 酉 유 |
| 월살(月殺) | 辰 진 | 丑 축 | 未 미 | 戌 술 |
| 망신살(亡身殺) | 巳 사 | 寅 인 | 申 신 | 亥 해 |
| 장성살(將星殺) | 午 오 | 卯 묘 | 酉 유 | 子 자 |
| 반안살(攀鞍殺) | 未 미 | 辰 진 | 戌 술 | 丑 축 |
| 역마살(驛馬殺) | 申 신 | 巳 사 | 亥 해 | 寅 인 |
| 육해살(六害殺) | 酉 유 | 午 오 | 子 자 | 卯 묘 |
| 화개살(華蓋殺) | 戌 술 | 未 미 | 丑 축 | 辰 진 |

> 예시: 年支가 午(인오술 그룹)이고 원국에 申이 있으면 → **역마살**. 같은 조건에서 일지 午 기준으로도 동일하게 적용.

### 11-3. 각 신살의 핵심 의미

| 신살 | 성격 | 핵심 해석 키워드 |
|---|---|---|
| 겁살(劫殺) | 흉살 | 갑작스러운 박탈·사고·재물 손실, 외부 충격 |
| 재살(災殺) | 흉살 | 구설·송사·감금, 외부 재앙과 관재(官災) |
| 천살(天殺) | 흉살 | 자연재해·불가항력적 손실, 기운 소진 |
| 지살(地殺) | 중립 | 이동·이사·출장, 역마에 준하는 이동수 |
| 년살(年殺) | 중립 | = 도화살(桃花殺). 이성 매력·인기·색욕·풍류 |
| 월살(月殺) | 흉살 | = 고초살(孤草殺). 고독·고초·끊임없는 시련 |
| 망신살(亡身殺) | 흉살 | 망신·수치·명예 손상, 스캔들 |
| 장성살(將星殺) | 길살 | 권위·지도력·리더십, 주도적 성취 |
| 반안살(攀鞍殺) | 길살 | 안정·관록·승진, 현재 자리 공고화 |
| 역마살(驛馬殺) | 중립 | 이동·여행·변동·해외, 활동 반경 확대 |
| 육해살(六害殺) | 흉살 | 방해·시비·손재, 주변인 배신 |
| 화개살(華蓋殺) | 중립 | 종교·예술·고독·정신세계, 영적 지향 |

### 11-4. 년지 기준 vs 일지 기준 해석 차이

| 기준 | 적용 범위 | 주요 해석 대상 |
|---|---|---|
| **년지(年支) 기준** | 사회적·환경적 흐름, 외부 세계와의 관계 | 직업·사회 활동·대외 인연·시대적 운명 |
| **일지(日支) 기준** | 개인 성향, 내면적·대인 관계 | 배우자·파트너·개인 습관·심리적 특성 |

- 두 기준에서 **같은 신살이 중복**으로 나타나면 해당 신살의 영향력이 매우 강하게 작용합니다.
- 원국의 지지에 신살이 없으면 대운·세운의 지지를 기준으로 유입 시점을 파악합니다.

## 12. 귀인(貴人) 및 기타 신살 — 위치별 분류

귀인·기타 신살은 **어떤 기준 글자(일간·월지·일주 전체)**로 찾느냐에 따라 먼저 구분하고,  
해당 신살이 **원국의 어느 기둥(년·월·일·시)**에 나타나는지에 따라 발현 영역이 달라집니다.

---

### 12-1. 일간(日干) 기준 귀인·신살 조견표

#### 천을귀인(天乙貴人) — 사주 최대 길신, 위기 시 구원의 손길

|---|---|
| 甲·戊·庚 | 丑·未 |
| 乙·己 | 子·申 |
| 丙·丁 | 亥·酉 |
| 壬·癸 | 卯·巳 |
| 辛 | 午·寅 |

#### 문창귀인(文昌貴人) — 학문·문서·언어 능력, 총명함

| 일간 | 甲 | 乙 | 丙 | 丁 | 戊 | 己 | 庚 | 辛 | 壬 | 癸 |
|---|---|---|---|---|---|---|---|---|---|---|
| 문창 지지 | 巳 | 午 | 申 | 酉 | 申 | 酉 | 亥 | 子 | 寅 | 卯 |

#### 학당귀인(學堂貴人) — 일간의 장생지(長生地), 배움·수련·자격 취득

| 일간 | 甲 | 乙 | 丙 | 丁 | 戊 | 己 | 庚 | 辛 | 壬 | 癸 |
|---|---|---|---|---|---|---|---|---|---|---|

#### 양인살(羊刃殺) — 일간의 겁재(劫財) 위치, 강렬한 승부욕·충동·사고수

| 일간 | 甲 | 乙 | 丙 | 丁 | 戊 | 己 | 庚 | 辛 | 壬 | 癸 |

- 관성(정관·편관)이 함께 있으면 양인이 억제되어 오히려 강한 추진력으로 발현됩니다.

> 두 귀인이 동시에 있으면 **천덕월덕(天德月德)**이라 하여 길함이 배가됩니다.

### 12-3. 일주(日柱) 전체 기준 특수 신살

#### 괴강살(魁罡殺) — 극단적 성격, 총명하나 고집 강함·부침 심함

| 해당 일주 |
|---|
| 庚辰·庚戌·壬辰·戊戌 |

- 관성과 충이 없으면 탁월한 지도력·영향력으로 발휘됩니다.
- 충·형이 가해지면 큰 기복·사고수로 나타납니다.

#### 백호대살(白虎大殺) — 혈광(血光)·수술·사고·갑작스러운 변고

| 해당 일주 |
|---|
| 甲辰·乙未·丙戌·丁丑·戊辰·壬戌·癸丑 |

- 원국에 있으면 직업상 혈(血)과 관련(의사·군인·경찰 등)되거나, 본인 또는 가족의 수술·사고 경험이 생길 수 있습니다.
- 천을귀인과 동주(同柱)하면 흉이 경감됩니다.

#### 원진살(怨嗔殺) — 원망·갈등·분리, 상대에 대한 지속적 감정 소모

- 일지와 월지, 또는 일지와 년지가 원진 관계이면 해당 육친과 평생 감정 갈등이 잠재합니다.
- 배우자 자리(일지)와 시지가 원진이면 부부·자녀 관계의 갈등.

#### 탕화살(湯火殺) — 화상·폭발·불의의 사고, 화(火) 재해

| 일지 | 寅 | 午 | 戌 | 亥 | 卯 | 未 | 申 | 子 | 辰 | 巳 | 酉 | 丑 |
|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 탕화 글자 | 巳 | 未 | 丑 | 申 | 子 | 辰 | 亥 | 卯 | 未 | 寅 | 午 | 戌 |

### 12-5. 원국 위치별 귀인·신살 해석 기준

어떤 귀인·신살이든 **원국의 어느 기둥(지지)**에 위치하느냐에 따라 영향받는 육친과 삶의 영역이 달라집니다.

| 위치 | 해당 육친·영역 | 귀인이 있으면 | 흉살이 있으면 |
|---|---|---|---|
| **년주(년간·년지)** | 조부모·가문·사회적 배경·초년 | 가문 덕, 사회적 후광, 초년 순탄 | 초년 고생, 가문 기반 취약 |
| **월주(월간·월지)** | 부모·형제·직업 기반·청장년기 | 부모 덕, 직업 귀인, 사회 진출 유리 | 부모 인연 박약, 직장 내 갈등·변동 |
| **일주(일간·일지)** | 본인·배우자·결혼·중년 | 배우자 덕, 본인 기운 강화, 대인 관계 원활 | 배우자 갈등·이별수, 본인 건강·사고 주의 |
| **시주(시간·시지)** | 자식·부하·사업 결실·말년 | 자식 덕, 말년 귀인, 사업 성취 | 자식 인연 박약, 말년 고독·사업 리스크 |

> **중요**: 귀인이 공망(空亡)에 걸리면 귀인의 효력이 반감됩니다.  
> 흉살이 합(合)·생(生)을 받으면 흉이 경감되며, 충(沖)·형(刑)을 받으면 흉이 가중됩니다.

## 13. 십이운성(十二運星) — 일간 대입 생사 주기

십이운성론은 **일간(日干)의 기운이 각 지지(地支)를 만났을 때 어느 생명 주기 단계에 있는지**를 나타냅니다.  
원국의 각 기둥 지지에 배속된 운성을 파악하면 해당 자리(배우자·직업·자식 등)의 에너지 강약과 성질을 파악할 수 있습니다.

### 13-1. 12단계 의미

| 순번 | 운성 | 에너지 | 핵심 키워드 |
|---|---|---|---|
| 1 | 장생(長生) | 상승 | 새 출발, 희망, 생명력 왕성, 귀인 만남 |
| 2 | 목욕(沐浴) | 상승 | 도화살과 중첩, 감수성·예능·색정, 불안정 |
| 3 | 관대(冠帶) | 상승 | 성장·준비·학업, 사회 진출 준비 |
| 4 | 건록(建祿) | 강 | 자립·독립·실력 발휘, 정관·비견과 유사 |
| 5 | 제왕(帝旺) | 최강 | 절정·성취·주도권, 지나치면 독선 |
| 6 | 쇠(衰) | 하강 | 안정 추구, 보수적, 진취성 약화 |
| 7 | 병(病) | 하강 | 건강 주의, 예민·섬세·예술적 감수성 |
| 8 | 사(死) | 약 | 정적(精的) 집중, 학문·종교·은둔 지향 |
| 9 | 묘(墓) | 저장 | 창고, 재물 보관, 집착·고집·매몰 |
| 10 | 절(絶) | 단절 | 허무, 변화 전 공백, 영적 민감 |
| 11 | 태(胎) | 잠재 | 새 씨앗, 계획·구상·잉태 단계 |
| 12 | 양(養) | 회복 | 의탁·보호·준비, 은인 관계 |

### 13-2. 일간별 지지 대입 조견표

양간(甲·丙·戊·庚·壬)은 **순행**, 음간(乙·丁·己·辛·癸)은 **역행**으로 배속합니다.

| 지지 | 甲 | 乙 | 丙·戊 | 丁·己 | 庚 | 辛 | 壬 | 癸 |
|---|---|---|---|---|---|---|---|---|
| 子 | 목욕 | 병 | 태 | 절 | 사 | 장생 | 제왕 | 건록 |
| 丑 | 관대 | 쇠 | 양 | 묘 | 묘 | 양 | 쇠 | 관대 |
| 寅 | 건록 | 제왕 | 장생 | 사 | 절 | 태 | 병 | 목욕 |
| 卯 | 제왕 | 건록 | 목욕 | 병 | 태 | 절 | 사 | 장생 |
| 辰 | 쇠 | 관대 | 관대 | 쇠 | 양 | 묘 | 묘 | 양 |
| 巳 | 병 | 목욕 | 건록 | 제왕 | 장생 | 사 | 절 | 태 |
| 午 | 사 | 장생 | 제왕 | 건록 | 목욕 | 병 | 태 | 절 |
| 未 | 묘 | 양 | 쇠 | 관대 | 관대 | 쇠 | 양 | 묘 |
| 申 | 절 | 태 | 병 | 목욕 | 건록 | 제왕 | 장생 | 사 |
| 酉 | 태 | 절 | 사 | 장생 | 제왕 | 건록 | 목욕 | 병 |
| 戌 | 양 | 묘 | 묘 | 양 | 쇠 | 관대 | 관대 | 쇠 |
| 亥 | 장생 | 사 | 절 | 태 | 병 | 목욕 | 건록 | 제왕 |

> 사용법: 일간이 **壬(임)**이고 월지가 **子**이면 → 월지 운성 = **제왕(帝旺)**  
> 직업 기반이 절정 에너지를 지닌 자리에 있어 강한 추진력과 성취를 나타냄.

### 13-3. 원국 기둥별 운성 해석

| 기둥 | 운성이 강할 때 (장생·관대·건록·제왕) | 운성이 약할 때 (사·묘·절·병) |
|---|---|---|
| **년지** | 가문 기운 강, 초년 환경 순탄, 조부모 덕 | 초년 어려움, 가문 기반 취약 |
| **월지** | 직업 에너지 충만, 사회 진출 유리, 부모 덕 | 직업 변동 잦음, 부모 인연 약함 |
| **일지** | 배우자 기운 강, 결혼 생활 안정 | 배우자 건강·인연 문제, 이별수 유의 |
| **시지** | 자식 에너지 강, 말년 결실 풍성 | 자식 인연 박약, 말년 외로움 |

### 13-4. 주요 운성 해석 심화

| 운성 | 원국 내 위치별 주의·활용 포인트 |
|---|---|
| **건록** | 일지에 건록 → 자수성가형. 월지에 건록 → 직업 독립성 강. 시지에 건록 → 말년 안정 |
| **제왕** | 월지 제왕 → 기세 넘침, 독선 주의. 일지 제왕 → 배우자가 강하거나 내가 주도 |
| **목욕** | 도화(桃花)의 성질과 겹침. 예술·연예·감성 분야 적성. 이성 인연 복잡 가능 |
| **묘** | 재물 보관력(저축 능력) 있음. 일지 묘 → 배우자가 소극적·집착적 경향 |
| **절** | 공망과 유사한 허전함. 절에서 태로 전환 시 새 출발 가능성 |
| **양** | 타인의 보살핌 속 성장. 의탁·종교·교육 관련 직업과 잘 맞음 |
